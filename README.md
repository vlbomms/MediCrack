# Medicrack
Spin-off of Quail


Intent: A general purpose question bank to be used as a starting kit to solve all non-essential elements of implementing a multiple-choice question bank (i.e. auth, question, security, highlighting, cross-outs, split-pane for passage and questions, etc...).

Current purpose/use: Transitioning Medcelerates app for MCAT from Flutter to Electron
